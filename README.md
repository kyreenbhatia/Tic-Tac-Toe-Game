# 🎮 Tic-Tac-Toe Game (Python Tkinter)

# A simple and interactive Tic-Tac-Toe game built using Python and Tkinter.
# This project provides a graphical interface for two players to play the classic game.

# --------------------------------------------------
# 📌 FEATURES
# --------------------------------------------------
# - Interactive 3x3 grid using Tkinter
# - Two-player gameplay (X vs O)
# - Automatic win detection
# - Winning combination highlighted in green
# - Pop-up message announcing the winner
# - Turn indicator display

# --------------------------------------------------
# 🛠️ TECHNOLOGIES USED
# --------------------------------------------------
# - Python 3
# - Tkinter (built-in GUI library)

# --------------------------------------------------
# 🚀 GETTING STARTED
# --------------------------------------------------

# Check Python version
python --version

# --------------------------------------------------
# ▶️ RUN THE PROJECT
# --------------------------------------------------

# 1. Clone the repository
git clone https://github.com/your-username/tic-tac-toe.git

# 2. Navigate to the project directory
cd tic-tac-toe

# 3. Run the game
python tic_tac_toe.py

# --------------------------------------------------
# 🎯 HOW TO PLAY
# --------------------------------------------------
# - Two players take turns: X and O
# - Click on a box to place your symbol
# - First player to get 3 in a row (horizontal, vertical, diagonal) wins
# - Winning row is highlighted and a popup shows the winner
